## 🚀 Overview

MriShan Drive is a modern cloud-based file management platform inspired by Google Drive and OneDrive. It enables users to securely upload, organize, search, share, and manage files with AI-powered assistance.

---

## ✨ Features

- 📁 File Upload & Download
- 🗂 Folder Management
- ⭐ Favorites
- 🗑 Recycle Bin
- 🔐 Secure Vault
- 🤝 File Sharing
- 📜 Version History
- 💬 Comments
- 📊 Storage Analytics
- 🔍 Smart Search
- 🤖 Gemini AI Assistant
- 👨‍💼 Admin Dashboard
- 📈 Audit Logs
- 🖥 Server Monitoring
- 🔔 Notifications
- 📱 Responsive Design
